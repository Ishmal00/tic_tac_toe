# Interactive AI Career Counselor 

## Overview
This is a **web-based interactive career discovery app** that uses fun, short questions to guide users toward career paths that match their interests and strengths. Inspired by Akinator, it asks dynamic questions and provides personalized recommendations.

## Features
- Interactive Akinator-style question flow
- Fun, short, and engaging questions
- Tracks answers to generate career recommendations
- Clean HTML/CSS/JS UI
- Visual summary of career options (can include charts)
